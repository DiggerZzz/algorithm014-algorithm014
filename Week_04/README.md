## 总结
dfs为栈的形式，先进后出，所以优先考虑递归法，类似二叉树的后序遍历，这里可能会出现多叉的情况。bfs为队列形式，所以优先考虑队列迭代法。
```
二分摸板
public void binarySearch(int[]  data, int target) {
    int left = 0, right = data.length - 1;
    while(left <= right) {
        int mid = (left + right) /2 ;
        if(target < data[mid]) {
            //往前查找
            right = mid - 1;
        } else if(target > data[mid]) {
            //往后查找
            left = mid + 1;
        } else {
            //找到目标
        }
    } 
}
```
